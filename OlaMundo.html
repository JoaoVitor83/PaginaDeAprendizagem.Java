Sout

Introduzindo os comandos básicos em java, podemos destacar o primeiro a ser utilizado por todos os dev's, o System.out.println ou melhor dizendo, o Sout(Usado como atalho no NetBeans, como o Sysout para o eclipse) pode ser utilizado para "printar" informações, varrer arrays e listas, fazer cálculos e converter valores.

Note que no exemplo acima, temos um método público. Esse método é OBRIGATÓRIO para executar o programa, onde o mesmo também é estatico, sendo o Main da nossa aplicação.

  • Main ---- É um método associado à classe e não a um objeto específico da classe -- assim, ele é definido como um método estático. Adicionalmente, deve ser um método público para permitir sua execução a partir da máquina virtual Java.
  • Static ---- serve para referenciar todos aqueles atributos/métodos de classe, ou seja, que podem ser acessados diretamente da definição da classe, sem precisar instanciar nenhum objeto. Você não precisa instanciar um objeto pra acessar algum método dela.
  • void ---- Essa "palavra" específica que aquele método não deve ter um retorno
  • Public ---- É um dos modificadores de acesso que possiblita o acesso do método em qualquer classe ou pacote do seu programa

JOptionPane

O JOptionPane segue a mesma linha de raciocínio do Sout, porém ele faz parte de uma biblioteca gráfica do java, o SWING, onde podemos fazer de comandos básicos até avançados de uma maneira visual, tal como fique mais agradável para testes do desenvolvedor, ou até mesmo para um cliente no teste de um produto final.

Ele também pode ser usado para cálculos, organizar os textos, mostrar listas e coleções, além de você poder, (com uma configuração mais profunda), mudar o seu tamanho, borda, forma e estilo, Aqui vai um exemplo simples de concatenação:

JOptionPane.showMessageDialog(null, "um + um = "+ 1+1);

Pode-se ver que aconteceu uma coisa muito curiosa, os números envolvidos nessa continha básica, não somaram e sim se juntaram. Na aula sobre Operadores aritiméticos, iremos ver mais sobre o ocorrido e os vários cálculos que podemos fazer com os mesmos.
