Classe Scanner

Conceito

O conceito da classe Scanner nada mais é do que um scanner(drrrr) que pode ser usado para ler tipos "normais" e primitivos de variáveis, onde é preciso instanciar a classe para usa-la, podendo ler váriaveis de String, char, int, float e double.
Para usarmos essa classe devemos importar a mesma no nosso projeto. Também vale lembrar que o Scanner é uma dependência do pacote Util do Java

Declaração e uso

Como foi dito anteriormente, precisamos import a classe Scanner pra usar a mesma, para fazer isso é bem simples, segue o exemplo aí:

import java.Util.Scanner;

CharAt

Usado em conjunto com a nossa clase, utilizamos o charAt, método que nos possibilita ler um caractere específico de uma String.

"variável" = "Scanner.next().charAt();"
